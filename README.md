# Jasa Curhat

Landing page neubrutalism untuk layanan teman cerita online. Dibangun ulang dengan React, Vite, Framer Motion, dan micro-interactions supaya siap dipakai sebagai proyek portfolio.

## Stack

- React + Vite
- Framer Motion
- Lucide React
- CSS custom dengan responsive layout

## Jalankan Lokal

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
